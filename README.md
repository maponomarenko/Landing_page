# MYBIKE - Bike store landing page
This project translates the stylish Figma design for a modern bike store into a PixelPerfect, responsive and fast landing page. It is adapted to all screen sizes. The UI is enhansed with hover effects, smooth scrolling and a convenient navigation menu. Every button and form input is clickable.

# Live demo
Explore the site by the [DEMO LINK](https://maponomarenko.github.io/Landing_page)

# Figma design

Compare the implemented page with the [Figma design](https://www.figma.com/design/NZQAIydtHo5QkINyGLHNcq/BIKE-New-Version?node-id=0-1&p=f&t=hax5AQKRiByKpwet-0)

# Technologies used

HTML5 for page markup
SCSS for styling

Vite as a build tool
ESLint for code quality
Prettier for automatic formatting

# Getting started
1. Clone the repository:
git clone https://github.com/maponomarenko/Landing_page.git
cd project-name

2. Install dependencies:
npm install
or
yarn install

3. Run the project locally:
npm start
or
yarn start

# Features
1. Responsive design - added two breakpoints for responsiveness on different screen resolutions
2. Hover effects on UI elements for better user experience. Including hover effect with appearing phone number in the header element.
3. Menu transition for better site navigation.
